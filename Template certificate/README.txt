
How to use Generate certificate application

- Step 1: Coppy all folder Generate certificate UNDER C DRIVE.
- Step 2: Run excecute file Template certificate.exe to run application
- Step 3: Choose button Choose excel file to choose file has list of students
- Step 4: Choose folder want to storage all pdfs file after generate
- Step 5: Choose which students that you want to generate
- Step 6: Click OK button to generate. Waiting few time when application generate.
		  Until the windows explorer open the folder, which you choose before.

Run AccessDatabaseEngine.exe file under Driver folder then try again.
